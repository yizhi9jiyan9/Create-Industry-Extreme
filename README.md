## 世界观：
你所在文明的人们利用着一个叫***魔方***的物质，它拥有着无穷无尽的能量。
后来，星球发生的异样使人们发现，那就是这个星球的核心。星球的不稳定威胁着人们。人们想修复***魔方***，但为时已晚。不久，星球破碎，世界崩坏。你足够幸运的存活了下来，但因为崩坏来到了一个未知的维度当中...
